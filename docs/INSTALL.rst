kagenomise.orders Installation
------------------------------

To install kagenomise.orders using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``kagenomise.orders`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        kagenomise.orders

* Re-run buildout, e.g. with:

    $ ./bin/buildout
